```Haskell
main :: IO()
main = do
    let
        num = 3 :: Int

    print $ isThree num     -- True
        where
            isThree = (== 3)
```

上記のコードは全く問題無く動く。\
でも次のコードは動かない。

```Haskell
main :: IO()
main = do
    let
        num = 3 :: Int

    print $ isThree num
        where
            isThree = (== 3)

    print num
```

原因と対策
---

where節が関数呼び出しにぶら下がっているものと誤認していた。\
実際には関数定義に紐付いている様子。\

つまり、次のように書いているのと同じ。

```Haskell
main :: IO()
main = do
    let num = 3 :: Int
    print $ isThree num

    where
        isThree = (== 3)

    print num
```

**そりゃバグるわ。**\
もう少し角度からこの結論を検討してみる。

```Haskell
main :: IO()
main = do
    let num = 3 :: Int

    print $ isThree (num + 1)     -- False
    print $ isThree num           -- True
        where
            isThree = (== 3)

    --print num
```

想像したとおりだった。where節の適用対象を間違えること無かれ。
